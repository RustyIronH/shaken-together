Create `src/physics/engine.ts`:
- `createPhysicsEngine()`: Creates `Engine.create()` with `constraintIterations: 4`, `enableSleeping: true`, `positionIterations: 6`, `velocityIterations: 4`. Sets `engine.gravity` to REALISTIC_MODE defaults.
- `startEngine(engine)`: Creates a `Runner` with fixed delta `FIXED_TIMESTEP` and runs it.
- `stopEngine(runner)`: Stops the runner.
- Also register a `beforeUpdate` event that caps all body velocities to `MAX_VELOCITY` (prevents tunneling through walls per research pitfall #2).

Create `src/physics/ragdoll-config.ts`:
- `resolveRagdollConfig(size: DollSize)`: Takes RAGDOLL_PARTS and RAGDOLL_JOINTS, applies `SIZE_SCALES[size]` to all dimension and offset values, returns scaled configs. This keeps the factory function clean.

Create `src/physics/ragdoll.ts`:
- `createRagdoll(x, y, size, colorScheme)`:
  1. Call `resolveRagdollConfig(size)` to get scaled parts and joints
  2. Create a collision group with `Body.nextGroup(true)` (negative = no self-collision)
  3. Create all 14 bodies using `Bodies.circle` or `Bodies.rectangle` based on `shape` field, with `collisionFilter: { group }`, `friction: currentMode.bodyFriction`, `restitution: currentMode.bodyRestitution`, `render: { fillStyle: colorScheme.primary/secondary/head }` based on label
  4. Create all 13 constraints using `Constraint.create` with `length: 0` (pin joints), stiffness and damping from current mode
  5. Build a `RagdollInstance` with `bodies` Map (label -> Body), `constraints` Map (label -> Constraint), unique `id` (use incrementing counter or crypto.randomUUID)
  6. Return the RagdollInstance

Create `src/physics/constraints.ts`:
- Internal state: array of `{ bodyA: Body, bodyB: Body, minAngle: number, maxAngle: number, baseMin: number, baseMax: number, goofyMultiplier: number }`
- `registerAngleLimits(engine, ragdoll)`: For each ANGLE_LIMITS entry, find the corresponding bodies in ragdoll.bodies Map, create a runtime limit record. If this is the first call, also register the `Events.on(engine, 'beforeUpdate', enforceAngleLimits)` hook.
- `unregisterAngleLimits(ragdollId)`: Remove limits for a removed ragdoll.
- `enforceAngleLimits()`: Loop through all registered limits. For each: compute `relativeAngle = bodyB.angle - bodyA.angle`. If below min, `Body.setAngle(bodyB, bodyA.angle + minAngle)` and `Body.setAngularVelocity(bodyB, 0)`. Same for max.
- `updateAngleLimitsForMode(mode)`: Multiply base limits by `mode.angleLimitMultiplier` to widen/tighten.

Create `src/physics/world.ts`:

- `createBoundaries(width, height)`: Create 4 static `Bodies.rectangle` walls (top, bottom, left, right) with `BOUNDARY_THICKNESS`, `isStatic: true`, `restitution: 0.5`, `friction: 0.3`. Position them so they form an enclosed box.

- `createScene(engine, width, height)`:
  1. Call `createBoundaries` and add to `engine.world` via `Composite.add`
  2. Create a `SceneState` object
  3. Call `setDollCount(DEFAULT_DOLL_COUNT)` to spawn initial ragdolls
  4. Return the SceneState

- `addRagdoll(engine, scene, width, height)`:
  1. Pick next color scheme from DOLL_COLOR_SCHEMES (cycle through)
  2. Pick a size variant (cycle through small/medium/large for variety)
  3. Generate random x,y within boundaries (leaving margin for ragdoll size)
  4. Call `createRagdoll(x, y, size, colorScheme)`
  5. `Composite.add(engine.world, ragdoll.composite)`
  6. `registerAngleLimits(engine, ragdoll)`
  7. Push to scene.ragdolls

- `removeRagdoll(engine, scene)`:
  1. Pop last ragdoll from scene.ragdolls
  2. `Composite.remove(engine.world, ragdoll.composite)`
  3. `unregisterAngleLimits(ragdoll.id)`

- `setDollCount(engine, scene, count, width, height)`:
  1. Clamp count to [MIN_DOLL_COUNT, MAX_DOLL_COUNT]
  2. While scene.ragdolls.length < count: addRagdoll
  3. While scene.ragdolls.length > count: removeRagdoll
  4. Update scene.dollCount

- `resetScene(engine, scene, width, height)`:
  1. For each ragdoll, generate new random x,y
  2. Calculate offset from current position to new position
  3. For each body in ragdoll: `Body.setPosition(body, newPos)`, `Body.setVelocity(body, { x: 0, y: 0 })`, `Body.setAngularVelocity(body, 0)`, `Body.setAngle(body, 0)` (reset to default pose)

- `applyMode(engine, scene, mode)`:
  1. Set `engine.gravity` to mode.gravity values
  2. For each constraint in world (filter by label starting with 'joint_'): set stiffness and damping
  3. For each non-static body: set restitution and friction
  4. Call `updateAngleLimitsForMode(mode)`
  5. Update scene.currentMode
  6. Do NOT reset positions or velocities (live switch per user decision)
