  <name>Task 1: Hold-to-shake fallback button</name>
  <files>src/input/shake-button.ts, src/main.ts</files>
  <action>
    Create src/input/shake-button.ts:

    1. **createShakeButton(engine, scene)** function:
       - Creates an HTML button element with text "Hold to Shake" (or a shake icon)
       - Positioned at bottom-center of screen, above the canvas, with fixed positioning
       - Styled inline (following project pattern: Object.assign(element.style, {...})):
         - Large touch target (min 48x48px), rounded corners, semi-transparent background
         - Visible but not obstructing too much play area
         - z-index above canvas but below panel
       - Returns the button element (caller appends to DOM)

    2. **pointerdown handler** (on the button):
       - Start an interval (~100ms) that applies random gravity bursts:
         - Random angle each tick: `const angle = Math.random() * Math.PI * 2`
         - Random magnitude: `const mag = 3 + Math.random() * 7` (moderate to strong)
         - `engine.gravity.x = Math.cos(angle) * mag * scaleFactor * modeMultiplier`
         - `engine.gravity.y = Math.sin(angle) * mag * scaleFactor * modeMultiplier`
         - Wake sleeping bodies (same logic as shake-manager: iterate non-static bodies, Sleeping.set(body, false))
       - modeMultiplier = `scene.currentMode.name === 'goofy' ? SHAKE_CONFIG.goofyMultiplier : 1.0`
       - scaleFactor from SHAKE_CONFIG.gravityScaleFactor, normalized appropriately

    3. **pointerup / pointercancel / pointerleave handler**:
       - Clear the interval
       - Do NOT snap gravity back -- let the existing updateGravityLerp in the ticker handle smooth return

    4. **showShakeButton(button)** / **hideShakeButton(button)**: Show/hide via display style. Used by main.ts based on shake permission result.

    5. **Wire into main.ts**:
       - After `initShake` resolves, check the result:
         - If 'granted': do NOT show button (motion works)
         - If 'denied' or 'unsupported': create button, append to ui-root, show it
       - Import createShakeButton, showShakeButton from shake-button.ts

    Important: Use Pointer Events (pointerdown/pointerup/pointercancel) not mouse events, for touch compatibility. Set `touch-action: none` on the button. The button must NOT interfere with canvas touch/drag (it sits outside the canvas element).
  </action>
